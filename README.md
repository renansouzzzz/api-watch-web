# api-watch-web

Dashboard frontend para monitoramento de APIs.

## Tecnologias

- Next.js 16 — Framework React
- Tailwind CSS — Estilização
- NextAuth.js — Autenticação

## Como rodar

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

> Requer o backend [api-watch](https://github.com/renansouzzzz/api-watch) rodando em `localhost:5000`
